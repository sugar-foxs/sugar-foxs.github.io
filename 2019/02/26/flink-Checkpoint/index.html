жң¬ж–Үдё»иҰҒд»Ӣз»Қflinkзҡ„checkpointжңәеҲ¶гҖӮ

  • Flink е®№й”ҷжңәеҲ¶зҡ„ж ёеҝғе°ұжҳҜжҢҒз»ӯеҲӣе»әеҲҶеёғејҸж•°жҚ®жөҒеҸҠе…¶зҠ¶жҖҒзҡ„дёҖиҮҙеҝ«з…§гҖӮFlinkзҡ„checkpoint жҳҜйҖҡиҝҮеҲҶеёғејҸеҝ«з…§е®һзҺ°зҡ„,жүҖд»ҘеңЁflinkдёӯиҝҷдёӨдёӘиҜҚжҳҜдёҖдёӘж„ҸжҖқгҖӮ

    • checkpointз”ЁжқҘдҝқиҜҒд»»еҠЎзҡ„й”ҷиҜҜжҒўеӨҚгҖӮд»»еҠЎеӨұиҙҘеҸҜд»Ҙд»ҺжңҖж–°зҡ„checkpointжҒўеӨҚгҖӮ
    • checkpointжңәеҲ¶йңҖиҰҒдёҖдёӘеҸҜйқ зҡ„еҸҜд»Ҙеӣһж”ҫж•°жҚ®зҡ„ж•°жҚ®жәҗ(kafka,RabbitMQ,HDFSвҖҰ)е’ҢдёҖдёӘеӯҳж”ҫstateзҡ„жҢҒд№…еӯҳеӮЁпјҲHDFS,S3вҖҰпјүгҖӮ

      checkpointConfig

  • йҖҡиҝҮи°ғз”ЁStreamExecutionEnvironment.enableCheckpointing(internalпјҢmode)еҗҜз”ЁcheckpointгҖӮ internalй»ҳи®ӨжҳҜ-1пјҢиЎЁзӨәcheckpointдёҚејҖеҗҜпјҢmodeй»ҳи®ӨжҳҜEXACTLY_ONCEжЁЎејҸгҖӮ

  • еҸҜи®ҫзҪ®checkpoint timeout,и¶…иҝҮиҝҷдёӘж—¶й—ҙcheckpointжІЎжңүжҲҗеҠҹпјҢcheckpointз»ҲжӯўгҖӮй»ҳи®Ө10еҲҶй’ҹгҖӮ

  • еҸҜи®ҫзҪ®checkpointеӨұиҙҘд»»еҠЎжҳҜеҗҰд№ҹеӨұиҙҘпјҢй»ҳи®ӨжҳҜtrueгҖӮ

  • еҸҜи®ҫзҪ®еҗҢж—¶иҝӣиЎҢзҡ„checkpointж•°йҮҸпјҢй»ҳи®Өдёә1гҖӮ

barrier

  • е°ҶbarrierжҸ’е…ҘеҲ°ж•°жҚ®жөҒдёӯпјҢдҪңдёәж•°жҚ®жөҒзҡ„дёҖйғЁеҲҶе’Ңж•°жҚ®дёҖиө·еҗ‘дёӢжөҒеҠЁгҖӮBarrier дёҚдјҡе№Іжү°жӯЈеёёж•°жҚ®пјҢж•°жҚ®жөҒдёҘж јжңүеәҸгҖӮдёҖдёӘ barrier жҠҠж•°жҚ®жөҒеҲҶеүІжҲҗдёӨйғЁеҲҶпјҡдёҖйғЁеҲҶиҝӣе…ҘеҲ°еҪ“еүҚеҝ«з…§пјҢеҸҰдёҖйғЁеҲҶиҝӣе…ҘдёӢдёҖдёӘеҝ«з…§гҖӮжҜҸдёҖдёӘ barrier йғҪеёҰжңүеҝ«з…§ IDпјҢе№¶дё” barrier д№ӢеүҚзҡ„ж•°жҚ®йғҪиҝӣе…ҘдәҶжӯӨеҝ«з…§гҖӮBarrier дёҚдјҡе№Іжү°ж•°жҚ®жөҒеӨ„зҗҶпјҢжүҖд»ҘйқһеёёиҪ»йҮҸгҖӮеӨҡдёӘдёҚеҗҢеҝ«з…§зҡ„еӨҡдёӘ barrier дјҡеңЁжөҒдёӯеҗҢж—¶еҮәзҺ°пјҢеҚіеӨҡдёӘеҝ«з…§еҸҜиғҪеҗҢж—¶еҲӣе»әгҖӮ
  • Barrier еңЁж•°жҚ®жәҗз«ҜжҸ’е…ҘпјҢеҪ“ snapshot n зҡ„ barrier жҸ’е…ҘеҗҺпјҢзі»з»ҹдјҡи®°еҪ•еҪ“еүҚ snapshot дҪҚзҪ®еҖј n (з”ЁSnиЎЁзӨә)гҖӮдҫӢеҰӮпјҢеңЁ Apache Kafka дёӯпјҢиҝҷдёӘеҸҳйҮҸиЎЁзӨәжҹҗдёӘеҲҶеҢәдёӯжңҖеҗҺдёҖжқЎж•°жҚ®зҡ„еҒҸз§»йҮҸгҖӮиҝҷдёӘдҪҚзҪ®еҖј Sn дјҡиў«еҸ‘йҖҒеҲ°дёҖдёӘз§°дёә checkpoint coordinator зҡ„жЁЎеқ—гҖӮ
  • з„¶еҗҺ barrier з»§з»ӯеҫҖдёӢжөҒеҠЁпјҢеҪ“дёҖдёӘ operator д»Һе…¶иҫ“е…ҘжөҒжҺҘж”¶еҲ°жүҖжңүж ҮиҜҶ snapshot n зҡ„ barrier ж—¶пјҢе®ғдјҡеҗ‘е…¶жүҖжңүиҫ“еҮәжөҒжҸ’е…ҘдёҖдёӘж ҮиҜҶ snapshot n зҡ„ barrierгҖӮеҪ“ sink operator пјҲDAG жөҒзҡ„з»ҲзӮ№пјүд»Һе…¶иҫ“е…ҘжөҒжҺҘж”¶еҲ°жүҖжңү barrier n ж—¶пјҢе®ғеҗ‘ the checkpoint coordinator зЎ®и®Ө snapshot n е·Іе®ҢжҲҗгҖӮеҪ“жүҖжңү sink йғҪзЎ®и®ӨдәҶиҝҷдёӘеҝ«з…§пјҢеҝ«з…§е°ұиў«ж ҮиҜҶдёәе®ҢжҲҗгҖӮ

еҰӮдҪ•и§ҰеҸ‘checkpointпјҹ

  • еңЁз”ҹжҲҗEcecutionGraphиҝҮзЁӢдёӯдјҡжіЁеҶҢзҠ¶жҖҒзӣ‘еҗ¬еҷЁCheckpointCoordinatorDeActivatorпјҢиҙҹиҙЈзӣ‘еҗ¬jobзҠ¶жҖҒпјҢjobеҸҳдёәиҝҗиЎҢзҠ¶жҖҒж—¶пјҢжү§иЎҢstartCheckpointSchedulerж–№жі•е®ҡж—¶жү§иЎҢScheduledTriggerпјҢScheduledTriggerдёӯжү§иЎҢзҡ„жҳҜtriggerCheckpointж–№жі•гҖӮ
  • еңЁиҝӣиЎҢдёҖдәӣжқЎд»¶жЈҖжҹҘд№ӢеҗҺпјҢйҰ–е…Ҳдјҡжһ„йҖ еҮәдёҖдёӘPendingCheckpointе®һдҫӢпјҢз„¶еҗҺеҶҚж”ҫеҲ°йҳҹеҲ—йҮҢпјҢеҸӘжңүеҪ“jobmanagerж”¶еҲ°SinkTaskеҸ‘жқҘзҡ„checkpointдҝқеӯҳжҲҗеҠҹзҡ„ж¶ҲжҒҜеҗҺпјҢиҝҷдёӘPendingCheckpointжүҚдјҡеҸҳжҲҗCompletedCheckpointпјҢиҝҷжүҚд»ЈиЎЁдёҖж¬Ўcheckpointдҝқеӯҳж“ҚдҪңзңҹжӯЈзҡ„е®ҢжҲҗдәҶгҖӮ еҶҚеҫҖдёӢзңӢпјҢжңҖйҮҚиҰҒзҡ„ж–№жі•жҳҜеҗ‘executionsеҸ‘йҖҒдҝқеӯҳcheckpointзҡ„ж¶ҲжҒҜпјҢйҖҡзҹҘtaskmanagerиҝӣиЎҢжң¬ең°зҡ„checkpointдҝқеӯҳгҖӮ дёӢйқўзңӢзңӢexecutionsд»Һе“ӘжқҘ,дёӢйқўиҝҷж®өд»Јз ҒеңЁPendingCheckpointз”ҹжҲҗд№ӢеүҚжү§иЎҢпјҡ еҸҜд»ҘзңӢеҲ°executionдёӯзҡ„е…ғзҙ йғҪжҳҜд»ҺtasksToTriggerдёӯиҺ·еҫ—пјҢйӮЈt
    asksToTriggerжҳҜд»Җд№Ҳе‘ўпјҹеҸҜд»ҘиҝҪжәҜеҲ°ExecutionGraphдёӯпјҡ зңӢдәҶдёӢcollectExecutionVerticesж–№жі•пјҢе№¶жІЎжңүиҜҙжҳҺжқҘжәҗпјҢз»§з»ӯзңӢverticesToTriggerд»Һе“ӘйҮҢжқҘпјҢиҝҪжәҜеҲ°StreamingJobGraphGeneratorдёӯпјҡ е°ұжҳҜиҜҙtriggerVerticesжҳҜйӮЈдәӣжІЎжңүиҫ“е…Ҙзҡ„иҠӮзӮ№пјҢеҚіж•°жҚ®жәҗгҖӮ
  • з»јдёҠпјҢй…ҚзҪ®дәҶcheckpointпјҢjobзҠ¶жҖҒзӣ‘еҗ¬еҷЁеңЁзӣ‘еҗ¬еҲ°зҠ¶жҖҒдёәrunningж—¶пјҢдјҡејҖеҗҜе®ҡж—¶еҷЁпјҢе®ҡж—¶еҗ‘TaskManagerеҸ‘йҖҒTriggerCheckpointж¶ҲжҒҜгҖӮ
